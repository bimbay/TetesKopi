# Tetes Kopi - Website UMKM Kopi Lokal

## Deskripsi Proyek
Website ini adalah platform digital untuk **Tetes Kopi**, sebuah UMKM kopi lokal yang berkomitmen menghadirkan pengalaman kopi autentik dari petani-petani Nusantara. Website ini mengusung tagline "Ada Cinta Di Setiap Tetesnya" dan menyajikan informasi lengkap tentang produk dan layanan kopi manual brew premium.

## Fitur Utama

###  Homepage
- Hero section dengan tagline menarik
- Navigasi yang responsif dengan hamburger menu untuk mobile
- Scroll indicator dan animasi smooth scrolling

###  Menu Section
- **Manual Brew Premium** dengan 3 metode brewing:
  - **V60** - Pour over dengan clarity tinggi (Rp 25.000)
  - **French Press** - Full immersion dengan body rich (Rp 24.000) 
  - **AeroPress** - Kombinasi pressure dan immersion (Rp 28.000)
- Badge "Best Seller" dan "Recommended" pada produk unggulan

###  Varian Biji Kopi
Slider interaktif yang menampilkan 5 jenis biji kopi:
- **Arabika** - Rasa kompleks dan lembut
- **Robusta** - Kuat dengan kafein tinggi
- **Liberika** - Unik dengan aroma seperti nangka
- **Excelsa** - Fruity dan winey
- **Gayo** - Premium Indonesia dengan aroma herbal

###  Gallery
Galeri foto yang menampilkan:
- Proses brewing
- Perfect cup hasil akhir
- Biji kopi premium
- Suasana cozy atmosphere
- Latte art
- Peralatan profesional

## Instalasi dan Penggunaan

### Cara Menjalankan
1. Download atau clone semua file
2. Pastikan struktur folder sesuai dengan yang dibutuhkan
3. Buka file `index.html` di browser
4. Website siap digunakan!

### Requirements
- Browser modern (Chrome, Firefox, Safari, Edge)
- Koneksi internet untuk loading Google Fonts dan Font Awesome
- File CSS dan JavaScript harus berada di folder yang benar
